DOM

什么是JS：脚本,解析型,弱类型,基于对象,动态类型
JavaScript分三个部分:
(1)ECMAScript标准,JS的基本语法
(2)DOM (Document Object Model)文档对象模型–操作页面元素
(3)BOM (Browser Object Model) 浏览器对象模型–操作浏览器

设置标签中的文本内容

textContent属性:谷歌、火狐、IE8不支持
innerText属性:谷歌、火狐、IE8都支持，火狐低版本不支持

设置标签中的文本内容

textContent属性:谷歌、火狐、IE8不支持
innerText属性:谷歌、火狐、IE8都支持，火狐低版本不支持

节点及相关内容

使用方式：element.attr
节点的属性: (1)nodeType 节点类型 (2)nodeName 节点名 (3)nodeValue 节点值

nodeType: 1.标签 2.属性 3.文本
nodeName: 1.大写标签名 2.小写属性名 3.text内容
nodeValue: 1.标签null 2.属性节点属性值 3.文本节点文本内容

获取父级节点和父级元素
parentNode 父级节点 parentElement 父级元素
childNodes 子节点（标签，属性，文本都算） children 子元素

获取属性节点
getAttributeNode(“属性名”)

其他获取节点方式
firstChild 第一个子节点 firstElementChild 第一个子元素
lastChild 最后一个子节点 lastElementChild 最后一个子元素
previousSibling 前一个兄弟节点 previousElementSibling 前一个兄弟元素
nextSibling 后一个兄弟节点 nextElementSibling 后一个兄弟元素

兼容问题
IE中获取子节点和兄弟节点，凡是获取节点的代码都是获取元素，获取元素的代码得到undefined

创建元素的方式

1.document.write(“标签及内容”) 缺点：在页面加载之后执行，会重新覆盖一个新页面，而在页面加载中创建正常
2.对象.innerHTML = “” 重写DOM结构,当前元素不再是之前的那个元素 要重新获取
3.document.createElement(“tagName”)

追加子元素和插入元素
element.appendChild(“tagName”) 追加子元素
element.insertBefore(“新元素”，”目标元素”) 目标元素前插入新元素
replaceChild(“新元素”,”目标元素”) 目标元素替换成新元素
removeChild(“目标元素”) 移除目标元素

元素绑定事件

方法一：element.on = function(){},重复绑定会被覆盖
方法二：element.addEventListener(“”,fn),事件监听,第三个参数决定是(false)冒泡还是捕获(true),默认冒泡 同个元素可重复绑定监听,IE8不支持
兼容：attachEvent(“on+”,fn) 只有IE8可用,事件类型有on,没有第三个参数

元素解绑事件

方法一：obj.on = null;
方法二：obj.removeEventListener(“”,); 事件类型和函数都要相同
方法三：obj.detachEvent(“”,fn); 事件类型和函数都要相同

事件冒泡

什么是冒泡：多个元素嵌套，有层次关系，这些元素被注册相同事件，如果里面元素事件被触发，那么外面元素的该事件自动触发

阻止事件冒泡
window.event.cancelBubble = true; IE特有
e.stopPropagation(); 谷歌和或火狐支持

事件阶段

详细解释——>事件阶段详解

DOM

浏览器对象模型，操作浏览器

浏览器顶级对象：window
页面顶级对象:document
页面所有内容都属于浏览器，页面中的内容都是window的，所以window可以省略

window下的特殊属性：
window.name 获取/设置窗口名字 top关键字等于window

扩展事件
window.onunload 页面关闭之后触发 window.onbeforeunload 页面关闭之前触发

History对象

对象包含用户（在浏览器窗口中）访问过的 URL。

###offset系列（宽高包括width,height,padding,border）(边距距离受父级元素margin影响)
offsetWidth 元素宽 offsetHeight 元素高 offsetLeft 距离视口左边距离 offsetTop 距离视口上面距离
详细解释——>offset系列

###Scroll系列（滚动区）
scrollWidth scrollHeight （元素内部实际内容宽高，没内容就是元素宽高）

scrollLeft scrollTop (向左向上滚动出去的距离，父级元素设置overflow:scroll）
补充：obj.onscroll 当鼠标滚动滚条是触发事件

详细解释——>Scroll系列

###client系列（可视区）
clientWidth clientHeight(可视区域宽高，不包含边框）

clientLeft clientTop (左边边框宽度）
补充：obj.onscroll 当鼠标滚动滚条是触发事件

详细解释——>client系列

###获取元素CSS样式值问题
兼容性问题：

ele.currentStyle //返回CSS对象，Firefox不支持

###阻止事件冒泡和阻止浏览器默认事件
阻止冒泡:
window.event.cancelBubble = true; / e.stopPropagation();

阻止浏览器默认事件：
return false; / e.preventDefault();
