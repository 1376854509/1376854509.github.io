DOM

什么是JS：脚本,解析型,弱类型,基于对象,动态类型
JavaScript分三个部分:
(1)ECMAScript标准,JS的基本语法
(2)DOM (Document Object Model)文档对象模型–操作页面元素
(3)BOM (Browser Object Model) 浏览器对象模型–操作浏览器

设置标签中的文本内容

textContent属性:谷歌、火狐、IE8不支持
innerText属性:谷歌、火狐、IE8都支持，火狐低版本不支持

设置标签中的文本内容

textContent属性:谷歌、火狐、IE8不支持
innerText属性:谷歌、火狐、IE8都支持，火狐低版本不支持

节点及相关内容

使用方式：element.attr
节点的属性: (1)nodeType 节点类型 (2)nodeName 节点名 (3)nodeValue 节点值

nodeType: 1.标签 2.属性 3.文本
nodeName: 1.大写标签名 2.小写属性名 3.text内容
nodeValue: 1.标签null 2.属性节点属性值 3.文本节点文本内容

获取父级节点和父级元素
parentNode 父级节点 parentElement 父级元素
childNodes 子节点（标签，属性，文本都算） children 子元素

获取属性节点
getAttributeNode(“属性名”)

其他获取节点方式
firstChild 第一个子节点 firstElementChild 第一个子元素
lastChild 最后一个子节点 lastElementChild 最后一个子元素
previousSibling 前一个兄弟节点 previousElementSibling 前一个兄弟元素
nextSibling 后一个兄弟节点 nextElementSibling 后一个兄弟元素

兼容问题
IE中获取子节点和兄弟节点，凡是获取节点的代码都是获取元素，获取元素的代码得到undefined

创建元素的方式

1.document.write(“标签及内容”) 缺点：在页面加载之后执行，会重新覆盖一个新页面，而在页面加载中创建正常
2.对象.innerHTML = “” 重写DOM结构,当前元素不再是之前的那个元素 要重新获取
3.document.createElement(“tagName”)

追加子元素和插入元素
element.appendChild(“tagName”) 追加子元素
element.insertBefore(“新元素”，”目标元素”) 目标元素前插入新元素
replaceChild(“新元素”,”目标元素”) 目标元素替换成新元素
removeChild(“目标元素”) 移除目标元素
